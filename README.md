# TiDB Wasm Markdown

Enjoy Tidb wasm on markdown


# Install

```sh
git clone https://github.com/imiskolee/tidb-wasm-markdown.git
cd tidb-wasm-markdown
python -m SimpleHTTPServer 8000
```

open `http://localhost:8000/example.html`
